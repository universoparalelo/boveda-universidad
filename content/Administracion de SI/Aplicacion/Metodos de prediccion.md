### Regresion lineal
Para datos que muestran una tendencia.
Se basa en una ecuacion de una variable dependiente de otra independiente.
y = ax + b
No estacionales.
### Medias moviles
Para datos que muestran un comportamiento estacionario.
No presenta mayores variaciones a corto plazo.
No estacionales, ni tendencia marcada. 
Es un promedio de una n cantidad de datos anteriores al dato a calcular, con un n, que minimice el error de prediccion, y se ajusta segun el n que de un menor error cuadratico.
### Suavizado exponencial
No estacionales y que no muestran una tendencia sistematica.
Similar al de Medias Moviles pero pone mas peso en las observaciones recientes.
Tiene un argumento alpha que varia entre 0 y 1, que indica cuanto valor se le da al dato observado mas reciente, y mientras mas alto mas sensible es a los cambios recientes.
Patrones estacionales.
